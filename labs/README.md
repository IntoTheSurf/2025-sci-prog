# Labs Directory

This directory contains laboratory exercises and hands-on activities for the Scientific Programming course.

## Structure

- `01.md` - Introduction to Scientific Programming
- `02.md` - (Upcoming) Data Structures and Algorithms
- `03.md` - (Upcoming) Numerical Computing with NumPy
- Additional lab files will be added as the course progresses

## Usage

Each lab file contains:
- Learning objectives and prerequisites
- Step-by-step exercises and problems
- Code examples and explanations
- Challenges and extensions
- Submission guidelines

## Guidelines

- Complete labs in numerical order
- Read the entire lab before starting
- Test your code thoroughly
- Document your solutions and thought process
- Ask questions and seek help when needed
- Submit completed labs according to course guidelines

## Development Environment

All labs are designed to be completed within the dev container environment, which includes:
- Python 3.13 with scientific computing libraries
- Jupyter notebooks for interactive development
- Code formatting and linting tools
- Integrated development environment support
